# Savians Assessments Frontend

Next.js App Router application for `assessments.savians.com`.

## Commands

```bash
npm install
npm run dev
npm run lint
npm run typecheck
npm run test
npm run build
```

Copy `.env.local.example` to `.env.local` for local development. Only values prefixed with `NEXT_PUBLIC_` are available to browser code; no QuickBooks, database, Resend, token-signing, or AWS secret belongs here.

See `../docs/PHASE_1_PROGRESS.md` for the living implementation status.
